在玩hexo的时候遇到的一点git使用技巧

rm -rf .git

这条命令删除.git文件

git push origin hexo

本地分支需切换到与远程分支相同，push时指定远程分支

部署hexo时，如果想本地源码push上githubs上，.git不能删

博客所有文件以及配置都在本机上，需要把本机上的所有文件以及配置都部署到在线仓库
